# Tuberculosis Detection Using Deep Learning

## Project Overview
This project presents a Deep Learning-based Tuberculosis Detection system using Chest X-ray images. The system uses Convolutional Neural Networks (CNN) for automatic classification of Tuberculosis-positive and normal chest X-rays.

The model is trained using TensorFlow and Keras and evaluated using multiple performance metrics such as Accuracy, Precision, Recall, F1-Score, and ROC-AUC.

---

## Features
- CNN-based medical image classification
- Chest X-ray preprocessing
- Binary classification (TB / Normal)
- Performance visualization
- Confusion matrix and ROC analysis
- TensorFlow/Keras implementation
